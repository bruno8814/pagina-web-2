Guía para Crear tu Primera Página Web

Aprende los conceptos básicos de HTML y CSS para construir tu primer sitio web

Paso 1: ¿Qué Necesitas para Empezar?

Para crear una página web básica, necesitas:

  • Un editor de texto: como Visual Studio Code, Sublime Text o Notepad.
  • Un navegador web: como Google Chrome o Firefox para ver tu página.
  • Conocimientos básicos de HTML y CSS: que aprenderemos a continuación.

Paso 2: Crear un Archivo HTML

Abre tu editor de texto y crea un nuevo archivo. Llámalo index.html. Este archivo contendrá la estructura de tu página web.

Estructura Básica de un Archivo HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>¡Hola, Mundo!</h1>
    <p>Esta es tu primera página web</p>
  </body>
</html>

Guarda el archivo y ábrelo en tu navegador para ver el resultado.

Paso 3: Entender las Etiquetas HTML

HTML utiliza etiquetas para definir cada parte de la página. Algunas etiquetas importantes son:

  • <h1>...</h1>: Encabezado principal de la página.
  • <p>...</p>: Párrafo de texto.
  • <a>...</a>: Enlace a otra página o sitio.
  • <img>: Imagen en la página.

Ejemplo de un enlace y una imagen:

<a href="https://www.w3schools.com/html/html_intro.asp">Visita Ejemplo</a>
<img src="https://www.w3schools.com/html/img_chrome.png" alt="Ejemplo de imagen">

Paso 4: Agregar Estilos con CSS

CSS permite agregar estilos a tu página para mejorar su apariencia. Crea un nuevo archivo en tu editor llamado styles.css y enlázalo en el archivo HTML, así:

<link rel="stylesheet" href="styles.css">

Ahora, en el archivo styles.css, agrega estilos como colores y tipos de fuente:

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

h1 {
    color: #4CAF50;
    text-align: center;
}

p {
    font-size: 16px;
    color: #333;
}

Guarda y recarga tu página en el navegador para ver los cambios de estilo.

Paso 5: Crear una Estructura Completa de Página

A continuación, aquí tienes un ejemplo completo de código HTML y CSS que puedes usar como modelo:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Página Completa</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <header>
      <h1>Bienvenido a Mi Página Completa</h1>
    </header>
    <section>
      <h2>Introducción</h2>
      <p>Esta es una página web simple para aprender HTML y CSS.</p>
    </section>
  </body>
</html>

Paso 6: Practicar y Explorar Más

¡Felicidades! Has creado tu primera página web básica. A medida que practiques, explora más etiquetas HTML y propiedades de CSS para mejorar tus habilidades.

Algunas ideas para practicar:

  • Añadir listas y tablas en HTML.
  • Utilizar colores y fondos con CSS.
  • Crear una estructura de múltiples páginas.
